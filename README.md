# How to use

### Download the latest released UF2 file
### Hold down bootsel on the sprig and plug it in while doing so
### Drag and drop the UF2 file onto the device
### Format a SD Card to Fat32 and put roms you own legally onto it in a .GB format
### Insert SD Card into the slot on the back of the screen
### Start the sprig and if done correctly then you should have your games listed!
